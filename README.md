# 🚀 Collaborative Document Editor

A real-time collaborative document editor built using MERN stack and Socket.io.

## ✨ Features
- 🔐 JWT Authentication
- ⚡ Real-time editing (Socket.io)
- 📄 Create & manage documents
- 🛡 Protected routes
- 🌙 Clean modern UI

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: Socket.io

## ⚙️ Installation

### 1. Clone repo
git clone https://github.com/dhanendiran/collaborative-doc-editor.git

### 2. Backend
cd server
npm install
nodemon server.js

### 3. Frontend
cd client
npm install
npm run dev
